## Реализовать операции циклического сдвига влево и вправо

**Циклический сдвиг** — это операция, при которой элементы последовательности (массива, строки, битового регистра) перемещаются так, что элементы, покидающие один конец, немедленно появляются на другом конце. Последовательность замыкается в кольцо.

```typescript
// Биты отбрасываемые из-за переполнения слева дополняются справа
cyclicLeftShift(0b10000000_00000000_00000000_00000001, 1) === 0b00000000_00000000_00000000_00000011

// Биты отбрасываемые из-за переполнения cправа дополняются слева
cyclicRightShift(0b10000000_00000000_00000000_00000001, 2) === 0b01100000_00000000_00000000_00000000
```

### Циклический сдвиг влево

При сдвиге влево на `k` позиций:

- Первые `k` элементов перемещаются в конец последовательности
- Остальные элементы сдвигаются влево, сохраняя свой порядок

```
Исходно:     [A, B, C, D]
Сдвиг влево: [B, C, D, A]
```

### Циклический сдвиг вправо

При сдвиге вправо на `k` позиций:

- Последние `k` элементов перемещаются в начало последовательности
- Остальные элементы сдвигаются вправо, сохраняя свой порядок

```
Исходно:      [A, B, C, D]
Сдвиг вправо: [D, A, B, C]
```

### Частные случаи

#### Сдвиг на 0 позиций

Последовательность не изменяется.

#### Сдвиг на половину длины

Сдвиг влево на `n/2` и сдвиг вправо на `n/2` дают одинаковый результат (при чётном `n`).
