## Написать функции для кодирования и декодирования строк

Для схемы вашей схемы кодирования из [ДЗ 1](https://gist.github.com/kobezzza/6f2ebd9e8c0ad956b2a7d5c74c087695) нужно реализовать функцию кодирования строки из JS в поток байтов и наоборот.
Если ваша кодировка позволяет хранить в рамках одного байта больше одного символа, то используйте это.

```typescript
const bytes = MyEncoding.encode("Какая-то строка!"); // Uint8Array
console.log(MyEncoding.decode(bytes)); // "Какая-то строка!"
```
